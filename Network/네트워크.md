# 네트워크
[[노드]]들이 데이터를 공유할 수 있게하는 하나의 통신망.

# 인터넷
이 세상에 있는 수 많은 통신망 중의 가장 큰 통신망. 우리는 인터넷이라는 통신망을 통하여 웹 서비스를 이용한다.
### 네트워크의 분류
- 크기에 따른 분류
1. LAN 
	-  가까운 지역을 하나로 묶은 네트워크
2. WAN
	- 가까운 지역끼리 묶인 LAN과 LAN을 다시 하나로 묶은 것
3. MAN
- 연결 형태에 따른 분류
1. Start 형 - 중앙 장비에 모든 노드가 연결
2. Mesh 형 - 서로서로 그물처럼
3. Tree 형
4. 기타
