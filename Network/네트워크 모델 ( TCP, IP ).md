#### 네트워크 계층 모델 
- TCP/IP 계층
	- 현재의 인터넷에서 컴퓨터들이 서로 정보를 주고받는데 쓰이는 통신 규약
- 1계층~4계층까지 각각 네트워크 인터페이스, 네트워크, 전송, 응용으로 나뉜다.

#### OST 7계층
- 네트워크 통신을 체계적으로 다루는 ISO에서 표준으로 지정한 모델.
- 데이터를 주고받을 때 데이터 자체의 흐름을 각 구간별로 나눠 놓은 것.
- 네트워크 패킷들은 사실 TCP/IP에 더 맞는다.
